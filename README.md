# Hotel Promenade AI Intelligence

An AI-powered intelligence system for Hotel Promenade focusing on NLP-based analysis, sentiment detection, and intelligent information retrieval.

## Project Overview

This project combines data science, machine learning, and web technologies to provide comprehensive AI-driven insights for hotel operations and guest intelligence.


## Key Components

- **NLP Pipeline**: Sentiment analysis, topic modeling, and text processing
- **RAG System**: Retrieval-Augmented Generation for intelligent Q&A
- **Model Fine-tuning**: Custom model training for hotel-specific tasks
- **API Backend**: REST API for model inference and data
